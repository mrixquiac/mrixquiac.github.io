Simulación de la ecuación de onda

Método de diferencias finitas para modelar la propagación de ondas en medios con velocidad variable, con condiciones de contorno absorbentes y análisis de estabilidad.

El problema

La ecuación de onda escalar en un medio no homogéneo describe la propagación de perturbaciones en un campo donde la velocidad de fase varía con la posición. El caso de velocidad constante tiene solución analítica exacta (d'Alembert), pero en cuanto la velocidad depende de x o t, los métodos numéricos se vuelven necesarios.

El objetivo de este proyecto fue implementar un solucionador robusto para la ecuación de onda 1D con velocidad variable, entender los criterios de estabilidad del esquema elegido, e implementar condiciones de contorno que absorban las ondas en lugar de reflejarlas.

Método

El esquema de diferencias finitas de segundo orden en espacio y tiempo (esquema explícito estándar) es el punto de partida natural. La discretización de la segunda derivada espacial con diferencias centradas da un esquema condicionalmente estable, cuya condición de estabilidad es el criterio de Courant-Friedrichs-Lewy (CFL):

El paso temporal Δt debe satisfacer Δt ≤ Δx / max(c), donde c es la velocidad de fase. Si se viola este criterio, el error numérico crece sin cota.

Para las condiciones de contorno absorbentes utilicé capas perfectamente adaptadas (PML), que introducen una región amortiguadora en los bordes del dominio. La idea es que las ondas que llegan al borde sean atenuadas antes de poder reflejarse.

Estructura del código

El solucionador está organizado en tres módulos: uno para la configuración del dominio y los parámetros, uno para el avance temporal y uno para la visualización. El módulo principal es deliberadamente simple: 

import numpy as np
from wave_solver import WaveSolver
from pml         import apply_pml

# Configuración del dominio
nx, nt  = 500, 2000
dx, dt  = 0.01, 0.005            # dt < dx/max(c) → CFL OK

# Velocidad variable: aumenta linealmente
x = np.linspace(0, nx*dx, nx)
c = 1.0 + 0.5 * (x / x[-1])     # c ∈ [1.0, 1.5]

# Condición inicial: pulso gaussiano centrado
u0   = np.exp(-((x - 2.5)**2) / (2 * 0.1**2))
u_tm1 = u0.copy()

solver = WaveSolver(c, dx, dt, pml_width=20)
u = solver.run(u0, u_tm1, nt)

Resultados

El solucionador reproduce correctamente el comportamiento esperado: la onda se acelera al entrar en la región de mayor velocidad, la amplitud disminuye por conservación de energía y las condiciones PML eliminan más del 99% de la reflexión en los bordes.

La comparación con la solución analítica en el caso de velocidad constante muestra un error de segundo orden en el paso de discretización, consistente con el esquema utilizado.

Lo que aprendí

El criterio CFL no es solo una condición técnica de estabilidad — tiene un significado físico preciso: el paso temporal no puede ser mayor que el tiempo que tarda la onda en cruzar una celda. Cuando se viola, el esquema numérico intenta propagar información más rápido que la velocidad física del sistema, y eso es fundamentalmente incorrecto.

La implementación de PML me obligó a entender las ecuaciones de Maxwell en medios absorbentes, aunque el problema era acústico. La misma técnica funciona en ambos casos, lo que sugiere que hay algo más profundo sobre la estructura de las ecuaciones de onda.
