09 — Tipografía de artículo (prose)

15 de marzo de 2025 · 7 min de lectura

Título del Artículo de Blog

Extracto o bajada del artículo que da contexto sobre el contenido. Una o dos oraciones que funcionan como gancho para el lector.

Este es el párrafo de apertura del artículo. El primer párrafo establece el contexto y presenta el tema de forma clara. No hay adornos innecesarios, solo claridad y precisión en la comunicación de ideas.

Subtítulo de Sección

El contenido continúa con desarrollo de ideas. Los párrafos están bien espaciados para facilitar la lectura en pantalla. El interlineado generoso y el ancho máximo de 680px crean una experiencia de lectura cómoda.

Este código sirve como ejemplo de bloque de código dentro de un artículo:

def funcion_ejemplo(parametro):
    """Documentación de la función."""
    resultado = parametro * 2
    return resultado

Los bloques de código están claramente diferenciados del texto normal. El código inline también tiene su propio estilo distintivo pero sutil.

Las citas destacadas tienen un estilo diferenciado con borde izquierdo en color acento. Se usan para resaltar ideas importantes o citas textuales.

Subsección más pequeña

El contenido continúa con más desarrollo. La jerarquía tipográfica clara permite al lector escanear el contenido y encontrar rápidamente lo que busca.

Los elementos de énfasis fuerte y énfasis cursiva están disponibles para dar matices al texto cuando es necesario, pero se usan con moderación.
